# 技術コンサルサービス
Web黎明期のネットスケープWebサーバー構築、Active Directoryの初期サンプルシステム構築、
日本Xbox事業部立ち上げ、社会人大学院でのMOT学習、欧州ビジネススクール留学などの経験を
活かし、以下の業務領域などの技術コンサルを行います。
- 古いシステムのクラウド化
- まったく新しいクラウドサービスの設計
- 画像認識AI
- データ解析 (Webログによる顧客解析)

弊社は北米・欧州の展示会にも出展しており、日本特有の事情を海外視点で見つめなおす事を
得意としております。

# 試作設計開発サービス
弊社はMicrosoft Azureのクラウドサービスプロバイダー (CSP) 認定を受けており、Azureを
使ったクラウドサービスの試作設計開発を得意としております。  
また、3Dプリンター、FPGA、基盤制作などを含む組み込みIoTの開発も行っており、クラウド
と連携したシステムの試作設計開発も行っております。

- 画像認識AIによる事業所入退出の管理
- 畜産物の生育中の行動量推定
- 物品識別によるPOS連携
- 大容量デジタルアーカイブ中の画像検索

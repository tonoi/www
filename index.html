クラウド処理の一部を端末上で分散処理するEdge Computing は、プライバシー保護、通信帯域節約、高レスポンスがメリットです。
一方、分散された機器の管理コストが問題となります。

Edge Computing と DevOps、双方理想的な手法ながら両立できませんでした。弊社が開発する Hybrid Computing はこの矛盾を解決します。

端末上のアプリはユーザーによる管理が必要ですが、クラウド上のサービスは開発者が直接コードを編集し利用(運用)しながら更新(開発)ができます。クラウドによってはじめて実現されるこの手法を DevOps と呼びます。
